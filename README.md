# CÍRCULO DT - Página Web Oficial

**Un grupo de estudiantes unidos por la amistad, el respeto y los mejores recuerdos.**

Página web premium, moderna y profesional para CÍRCULO DT - I.E.P. Diego Thomson.

## 🎨 Características

- ✨ Diseño premium y moderno inspirado en Apple, Spotify y Discord
- 🎬 Animaciones suaves con Framer Motion
- 📱 Responsive Design - Mobile First
- 🖼️ Galería colaborativa con fotos, likes y comentarios
- 📢 Panel de anuncios dinámico
- 📖 Libro de recuerdos
- 👥 Sección de integrantes con buscador
- 🔐 Panel administrador funcional
- 🌙 Tema oscuro elegante
- ☁️ Base de datos en Supabase

## 🛠️ Tecnologías

- React 18
- Tailwind CSS
- Framer Motion
- Supabase
- React Router
- Vite

## 📁 Estructura

```
src/
├── components/
├── pages/
├── hooks/
├── services/
├── App.jsx
└── main.jsx
```

## 🚀 Instalación

```bash
npm install

```
